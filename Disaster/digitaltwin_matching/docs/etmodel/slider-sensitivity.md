# 슬라이더 비교 분석 도구

> 한 줄 요약: 특정 입력 세트가 CO₂, 비용 등 결과 지표에 미치는 민감도를 정량적으로 비교합니다.

## 무엇을 할 수 있나
- 선택한 슬라이더 묶음의 변경이 결과 지표에 미치는 영향을 자동으로 계산
- 시나리오 간 영향도 비교를 통해 우선 조정해야 할 입력을 식별

## 언제/왜 쓰나
- 정책·투자 시나리오에서 어떤 입력이 목표에 가장 큰 영향을 주는지 파악할 때
- 슬라이더 조합 최적화, 민감도 분석 리포트를 제작할 때

## 사용 방법 (Step-by-step)
1. 기준 시나리오를 선택하고 비교 대상 슬라이더 세트와 결과 지표 쿼리를 지정합니다.
2. 분석을 실행해 각 슬라이더 변경에 따른 ΔCO₂, Δ비용, Δ발전믹스를 계산합니다.
3. 영향도가 큰 항목을 기준으로 시나리오를 재조정하거나 보고서를 작성합니다.

## 결과 해석 (주요 지표/차트)
- 변화량 표와 순위 차트에서 입력별 Δ지표를 확인하고 우선순위를 설정합니다.

## 주의사항
- 비교 대상 슬라이더가 상호배제/자동밸런싱 그룹에 속한 경우 결과 해석 시 제약을 함께 검토합니다.
- 분석 실행 전 기준 시나리오의 슬라이더 상태를 잠그거나 내보내 백업해 둡니다.
