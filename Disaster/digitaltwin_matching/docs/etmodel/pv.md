# 전력 부문: 태양광(PV)

> 한 줄 요약: 태양광 보급률과 출력 프로파일 가정을 통해 발전믹스와 계통 유연성 요구를 평가합니다.

## 무엇을 할 수 있나
- 분산형/중앙집중형 PV 비중과 연도별 누적 설치용량을 설정
- 용량계수, 출력 프로파일을 조정해 발전량과 계통 부하 곡선을 비교
- 잉여 전력·피크 수요 데이터를 기반으로 저장·수요반응 전략을 평가

## 언제/왜 쓰나
- 지자체/국가의 태양광 보급 목표(예: 2030, 2040)를 시나리오에 반영할 때
- 저장장치·수요반응·전력망 강화 필요성을 사전에 검토할 때

## 사용 방법 (Step-by-step)
1. 시나리오에서 전력 부문 → 태양광 슬라이더로 이동합니다.
2. 누적 설치용량(GW)과 분산/집중 비중(%), 용량계수(%)를 설정합니다.
3. 발전믹스·시계열 차트에서 잉여·부족 전력 패턴과 CO₂ 감축 기여도를 확인합니다.

## 결과 해석 (주요 지표/차트)
- 연간 발전량(TWh), 정시별 잉여·부족 전력, 덕커브 패턴, CO₂ 감축 기여도를 분석합니다.

## 주의사항
- 풍력·원자력 등 타 발전원과 자동 밸런싱 제약으로 인해 결과가 상호 영향받을 수 있습니다.
- 수요 효율화 또는 전기화 가정이 바뀌면 필요한 PV 용량이 달라집니다.
