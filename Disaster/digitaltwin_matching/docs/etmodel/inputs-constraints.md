# 입력과 제약 가이드

> 한 줄 요약: 자동 밸런싱과 상호배제 규칙을 이해하면 일관된 시나리오 입력을 빠르게 구성할 수 있습니다.

## 무엇을 할 수 있나
- 합계 제약이 있는 슬라이더 그룹에서 자동 밸런싱 동작을 파악하고 활용
- 상호배제 입력을 관리해 충돌 없는 시나리오를 설계

## 언제/왜 쓰나
- 다수의 발전원·수요 항목을 동시에 조정해 믹스를 설계할 때
- 시나리오 비교·민감도 분석을 위해 입력 변경 이력을 관리할 때

## 사용 방법 (Step-by-step)
1. 자동 밸런싱 그룹에서는 한 항목만 조정하고 나머지는 자동 조정에 맡깁니다.
2. 상호배제 항목은 비활성화 상태를 확인하고 필요한 경우 다른 옵션을 초기화합니다.
3. 변경 이력을 내보내거나 버전 관리로 기록해 비교 분석에 활용합니다.

## 결과 해석 (주요 지표/차트)
- 입력 요약 테이블, 자동 밸런싱 로그, 민감도 결과에서 적용된 제약을 확인합니다.

## 주의사항
- 합계 100% 제약 그룹에서 수동으로 여러 항목을 동시에 조정하면 오차가 발생할 수 있습니다.
- 다른 부문(수요 절감 등) 변경에 따라 발전원 자동 밸런싱 결과가 달라짐을 염두에 둡니다.
