# ETModel 상세 페이지 묶음

## 사용 가이드
1. 신규 상세 페이지가 필요하면 `template.md`를 복사해 내용을 채웁니다.
2. 문서에 명시된 이미지 자리표시자를 실제 스크린샷으로 교체합니다.
3. `README.md`에 새 문서를 추가해 탐색성을 유지합니다.
